# 단어 학습 플래시카드 앱

## 프로젝트 소개

단어 학습을 위한 플래시카드 웹 애플리케이션입니다. 단어를 먼저 보여주고 의미를 4지선다형으로 선택하는 형태로 학습할 수 있습니다.

## 주요 기능

### ✨ 핵심 기능
- **플래시카드 학습**: 단어를 먼저 보여주고 의미를 맞추는 방식
- **4지선다형 퀴즈**: 4개의 선택지 중 정답 선택
- **점수 기록**: 학습 점수를 저장하고 조회 가능
- **반응형 디자인**: PC와 모바일에서 모두 사용 가능

### 📚 학습 기능
- **랜덤 단어 선택**: 10개의 단어를 랜덤으로 선택하여 학습
- **진행률 표시**: 현재 진행 상황을 시각적으로 표시
- **즉각적 피드백**: 정답/오답 여부 즉시 표시
- **플래시카드 뒤집기**: 정답 확인 기능

### 📊 점수 관리
- **점수 계산**: 10점 만점으로 점수 계산
- **점수 기록**: 학습 이력 저장
- **평균 점수**: 전체 학습 평균 점수 표시
- **점수 조회**: 이전 학습 기록 조회

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 스타일링, 반응형 디자인
- **JavaScript (ES6+)**: 모던 JavaScript
- **Font Awesome**: 아이콘


## 사용 방법

### 1. 학습 시작
- 메인 화면에서 "학습 시작" 버튼 클릭
- 랜덤으로 10개의 단어가 선택됨

### 2. 단어 학습
1. 단어가 표시됨
2. 의미를 맞춰보세요!
3. 4개의 선택지 중 하나 선택
4. 정답 여부 즉시 표시
5. 플래시카드를 클릭하면 정답 확인 가능

### 3. 점수 확인
- 학습 완료 후 점수 확인
- 정답 개수, 오답 개수 표시
- 점수 기록 저장

### 4. 점수 기록 조회
- 상단의 "점수 기록" 버튼 클릭
- 이전 학습 기록 조회 가능

## 데이터 구조

### 단어 테이블
```javascript
{
  id: "1",
  word: "abundant", 
  meaning: "풍부한, 많은",
  difficulty: "easy"
}
```

### 점수 테이블
```javascript
{
  id: "uuid",
  score: 8,
  correct_count: 8,
  total_questions: 10,
  study_date: 1640000000000
}
```

## 화면 구성

### 1. 시작 화면
- 앱 제목
- 총 단어 수 표시
- 평균 점수 표시
- 학습 시작 버튼

### 2. 학습 화면
- 진행률 표시
- 플래시카드 (단어 표시)
- 4지선다형 선택지
- 현재 점수 표시

### 3. 결과 화면
- 최종 점수 표시
- 정답/오답 개수
- 다시 학습 버튼
- 처음으로 버튼

### 4. 점수 기록 화면
- 이전 학습 기록 목록
- 점수, 날짜, 정답률 표시
